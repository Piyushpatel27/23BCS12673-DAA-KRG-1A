Radix Sort

Radix Sort is a non-comparative sorting algorithm that sorts numbers by processing individual digits, from the least significant to the most significant.

How It Works

  • Find the maximum number to determine the number of digits.
  • Sort numbers based on each digit, starting from the least significant (rightmost) digit.
  • Use a stable sorting algorithm like Counting Sort for each digit placement.
  • Repeat the process for all digits until the array is sorted.

Time Complexity

Best Case: O(nk) (Where k is the number of digits in the largest number)

Worst Case: O(nk) (Stable sorting at each digit level)

Average Case: O(nk)
