Bubble Sort

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. The process repeats until the array is sorted.

How It Works

  • Compare adjacent elements in the array.
  • Swap them if they are in the wrong order.
  • Repeat the process for each element until the entire list is sorted.
  • The largest element "bubbles up" to its correct position after each pass.

Advantages of Bubble Sort

  • Simple and easy to understand.
  • Requires only a few lines of code.
  • Best case time complexity is O(n) when the array is already sorted.
  • No extra memory required (in-place sorting algorithm).

Disadvantages of Bubble Sort

  • Slow for large datasets due to O(nÂ²) worst-case complexity.
  • Performs unnecessary swaps even when the array is almost sorted.
  • Other algorithms like Quick Sort or Merge Sort are much more efficient.

Real-World Applications

  • Good for **small datasets** where simplicity matters.
  • Useful for **teaching sorting concepts** due to its easy implementation.
  • Can be used when **only a few swaps are needed** (e.g., nearly sorted lists).
  • Embedded systems or situations where **minimal code is required**.
