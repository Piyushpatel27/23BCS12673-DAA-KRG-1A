Insertion Sort

Insertion Sort is a simple and efficient sorting algorithm that builds the sorted array one item at a time, inserting each new element into its correct position.

How It Works

  • Start from the second element, considering the first element as sorted.
  • Compare the current element with the sorted elements and insert it at the correct position.
  • Repeat this process for all elements in the array.

Time Complexity

Best Case: O(n) (Already sorted input, minimal shifting)
