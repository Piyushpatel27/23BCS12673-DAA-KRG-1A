Merge Sort

Merge Sort is a divide-and-conquer sorting algorithm that recursively splits an array into two halves, sorts them, and then merges them back together in sorted order.

How It Works

  • Divide the array into two halves.
  • Recursively apply Merge Sort to each half.
  • Merge the two sorted halves into a single sorted array.

Time Complexity

Best Case: O(n log n)

Worst Case: O(n log n)

Average Case: O(n log n)

Example

        Input: [8, 3, 5, 2, 7, 6, 4]
