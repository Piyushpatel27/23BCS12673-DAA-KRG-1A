Bubble Sort Introduction

Bubble Sort:  It Is The Sorting Technique Which Sort The Element By Comparing It with the Adjacent Elements.

Working:  Repeatedly Compares the adjacent Element and swaps them if they are in the Wrong Order.

Time Complexity:  O(n2)In Best Case and Average Case.
                          O(n)If The Array Is Already Sorted.
