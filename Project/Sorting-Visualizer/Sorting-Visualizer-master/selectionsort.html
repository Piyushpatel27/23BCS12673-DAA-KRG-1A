Selection Sort

Selection Sort is a simple comparison-based sorting algorithm. It repeatedly finds the minimum element from the unsorted portion and swaps it with the first unsorted element.

How It Works

  • Find the smallest element in the unsorted section of the array.
  • Swap it with the first unsorted element.
  • Move to the next unsorted element and repeat the process.
