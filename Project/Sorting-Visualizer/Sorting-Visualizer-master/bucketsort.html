Bucket Sort

Bucket Sort is a sorting algorithm that distributes elements into multiple "buckets," sorts each bucket individually, and then combines them to get the final sorted array.

How It Works

  • Divide the elements into a number of buckets based on a predefined range.
  • Sort each individual bucket using another sorting algorithm (e.g., Insertion Sort or Merge Sort).
  • Concatenate all the sorted buckets to form the final sorted array.

Time Complexity

Best Case: O(n + k) (If elements are evenly distributed among buckets)

Worst Case: O(n¬≤) (If all elements go into one bucket, degrading to a slow sorting algorithm)

Average Case: O(n + k)
